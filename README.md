# **pie-salt-marsh-deep-core**
This study explores organic matter and microbial communities in deep (~3 meter) salt marsh sediments collected from the [Plum Island LTER](https://pie-lter.mbl.edu/) in 2016. Stay tuned for a pre-print and updated code for sequence and NMR spectroscopy analysis.

See [here](https://www.biorxiv.org/content/10.1101/2025.07.18.665577v2.abstract) for another pre-print associated with this dataset led by [Dr. Joseph Vineis](https://jvineis.github.io/)
